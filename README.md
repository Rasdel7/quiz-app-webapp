# Quiz App with Leaderboard 🎯

Interactive quiz with 32 questions across 4 categories,
live scoring, instant feedback and persistent leaderboard.

## Live Demo
[Click here](https://quiz-app-webapp-xvmmswsbucev7cwmbibtfa.streamlit.app)

## Features
- 4 categories: Data Science, Python, ML, General CS
- Random mix mode
- Instant feedback with explanations
- Persistent leaderboard with rankings
- Score distribution analytics
- Answer review after quiz

## Tools Used
- Python, Streamlit, Pandas, Matplotlib

## How to Run Locally
pip install streamlit pandas matplotlib
streamlit run app.py
